# Consulta CEP

Este repositório contém um pequeno projeto desenvolvido para praticar JavaScript, consumindo uma API de CEP para buscar e preencher automaticamente os dados de endereço a partir de um CEP informado pelo usuário.

## Objetivo do Projeto

O objetivo deste projeto foi exercitar conceitos básicos de JavaScript no front-end, incluindo:

- Manipulação do DOM
- Consumo de API via HTTP
- Tratamento de respostas em JSON
- Validação simples de entrada do usuário

## Funcionalidades

- Campo para digitar um CEP.
- Validação básica do CEP informado.
- Consulta de endereço em uma API pública de CEP.
- Preenchimento automático dos campos de endereço (rua, bairro, cidade, estado) quando o CEP é válido.
- Exibição de mensagem de erro quando o CEP é inválido ou não encontrado.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- API pública de CEP (como ViaCEP ou similar)

## Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/Tyago03/ConsultaCEP.git
````
